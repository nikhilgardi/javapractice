Java 8 Features

Feature Description Example
Lambda Expressions Provides a clear and concise way to represent one method interface using an expression. (a, b) -> a + b
Functional Interfaces Interfaces with a single abstract method (SAM), used with Lambda expressions. @FunctionalInterface interface MyInterface { void doSomething(); }
Default Methods Interface methods with default implementation. Allows extending interfaces without breaking the implementation. default void defaultMethod() { System.out.println("Default method"); }
Static Methods in Interfaces Interfaces can have static methods, which can be called directly from the interface without an object. interface MyInterface { static void staticMethod() { System.out.println("Static Method"); }}
Streams API Allows functional-style operations on streams of elements, like filter, map, and reduce. List nums = Arrays.asList(1, 2, 3); nums.stream().filter(n -> n > 1).forEach(System.out::println);
Optional A container object which may or may not contain a value. Helps in avoiding `NullPointerException`. Optional name = Optional.of("Java"); name.ifPresent(System.out::println);
Method References A shorthand syntax for calling a method using the `::` operator. List names = Arrays.asList("Alice", "Bob"); names.forEach(System.out::println);
New Date and Time API A new `java.time` package to handle date and time more effectively and accurately (replaces `java.util.Date`). LocalDate date = LocalDate.now();
Collectors Utility methods in the `Collectors` class for working with streams like `groupingBy`, `joining`, `partitioningBy`. List list = Arrays.asList("a", "b", "c"); String result = list.stream().collect(Collectors.joining(", "));
Nashorn JavaScript Engine A new lightweight, high-performance JavaScript engine to run JavaScript code from within Java programs. ScriptEngine engine = new ScriptEngineManager().getEngineByName("Nashorn"); engine.eval("print('Hello');");
Parallel Streams Allows parallel processing of streams for better performance on multi-core systems. nums.parallelStream().forEach(System.out::println);
Type Annotations Enhances the ability to add annotations to types, helping improve code quality, safety, and readability. @NotNull String name;
Repeatable Annotations Allows the same annotation to be applied multiple times to the same declaration. @Author(name="John") @Author(name="Jane") public class MyClass {}
Interface Inheritance Interfaces can now extend other interfaces with default methods, providing more flexibility. interface A { default void method() {} } interface B extends A {}
Streams API for IO New methods for working with I/O, like reading lines from files. Files.lines(Paths.get("file.txt")).forEach(System.out::println);
Enhancements to the java.util package Added new utility methods for collections, including `List`, `Set`, and `Map`. Map map = new HashMap<>(); map.putIfAbsent("key", 1);

Java 8 - Lambda Expressions

1. What is a Lambda Expression?

A lambda expression is a short block of code that takes in parameters and returns a value. Used to implement functional interfaces.

(int a, int b) -> a + b
2. Simple Example (Runnable)

Using a lambda to implement a Runnable:

Runnable r = () -> System.out.println("Hello from Lambda!");
r.run();
3. Functional Interface Example

Define and use a custom functional interface:

@FunctionalInterface
interface MathOperation {
    int operate(int a, int b);
}

MathOperation add = (a, b) -> a + b;
System.out.println(add.operate(5, 3));
4. Lambdas with Collections (forEach) 
List<String> list = Arrays.asList("Apple", "Banana", "Orange");
list.forEach(item -> System.out.println(item));
5. Lambda with Comparator 
List<String> names = Arrays.asList("Steve", "Amanda", "Brian");
names.sort((s1, s2) -> s1.compareTo(s2));
6. Lambda with Streams 
List<String> names = Arrays.asList("Ram", "Ravi", "Shyam", "Ramesh");

names.stream()
    .filter(name -> name.startsWith("R"))
    .map(String::toUpperCase)
    .sorted()
    .forEach(System.out::println);
7. Custom Functional Interface 
@FunctionalInterface
interface StringConcat {
    String combine(String s1, String s2);
}

StringConcat sc = (a, b) -> a.toUpperCase() + "-" + b.toLowerCase();
System.out.println(sc.combine("Hello", "WORLD"));
8. Tricky: Variable Scope in Lambda

Variables used inside lambda must be effectively final.

String prefix = "Hello ";
Consumer<String> c = s -> System.out.println(prefix + s);
c.accept("Java");

// prefix = "Hi "; // ❌ Not allowed!
9. Returning Lambda from a Method 
public static Function<Integer, Integer> getMultiplier(int factor) {
    return (n) -> n * factor;
}

Function<Integer, Integer> triple = getMultiplier(3);
System.out.println(triple.apply(5)); // 15
10. MNC-Level: Replace Anonymous Inner Class 
// Old way
Comparator<String> comp = new Comparator<>() {
    public int compare(String s1, String s2) {
        return s1.length() - s2.length();
    }
};

// Lambda
Comparator<String> compLambda = (s1, s2) -> s1.length() - s2.length();
11. MNC-Level: Lambda + Stream + Reduce 
List<Integer> nums = Arrays.asList(1, 2, 3, 4, 5, 6);

int sum = nums.stream()
              .filter(n -> n % 2 == 0)
              .map(n -> n * n)
              .reduce(0, (a, b) -> a + b);

System.out.println(sum); // Output: 56
12. MNC-Level: Lambda with Custom Objects 
class Employee {
    String name;
    int age;
    Employee(String name, int age) {
        this.name = name;
        this.age = age;
    }
}

List<Employee> list = Arrays.asList(
    new Employee("Ram", 25),
    new Employee("Shyam", 30),
    new Employee("Ravi", 22)
);

// Sort by age
list.sort((e1, e2) -> e1.age - e2.age);

Java 8 Functional Interfaces

1. What is a Functional Interface?

A Functional Interface is an interface that has exactly one abstract method. It can have any number of default or static methods. Lambda expressions and method references implement functional interfaces.

2. Basic Example of Functional Interface 
@FunctionalInterface
interface Greeting {
    void sayHello(String name);
}

// Lambda implementation
Greeting greet = name -> System.out.println("Hello " + name);
greet.sayHello("Alice"); // Output: Hello Alice
3. Intermediate Example - Function Interface

Java provides built-in functional interfaces in java.util.function package such as Function, Predicate, Consumer, and Supplier.

Function<Integer, String> intToString = i -> "Number: " + i;
System.out.println(intToString.apply(10));  // Output: Number: 10
4. Advanced Example - Custom Functional Interface with Default Method 
@FunctionalInterface
interface Calculator {
    int calculate(int a, int b);

    default int square(int a) {
        return a * a;
    }
}

Calculator add = (a, b) -> a + b;
System.out.println(add.calculate(5, 3));  // Output: 8
System.out.println(add.square(4));         // Output: 16
5. Complex Example - BiFunction and Function Chaining 
BiFunction<Integer, Integer, Integer> multiply = (a, b) -> a * b;
Function<Integer, Integer> square = x -> x * x;

// Compose functions
Function<Integer, Integer> multiplyThenSquare = multiply.andThen(square);

System.out.println(multiplyThenSquare.apply(3, 4)); // Output: 144
// Explanation: (3 * 4) = 12, 12^2 = 144
6. Tricky Example - Predicate with Negation and Composition 
Predicate<String> isNotEmpty = s -> !s.isEmpty();
Predicate<String> isNotBlank = s -> s.trim().length() > 0;

// Compose predicates
Predicate<String> validString = isNotEmpty.and(isNotBlank);

System.out.println(validString.test("  "));  // Output: false
System.out.println(validString.test("Java"));  // Output: true
7. MNC-Level Use Cases

Q1: Custom Functional Interface with Lambda, Default, and Static Methods

@FunctionalInterface
interface StringProcessor {
    String process(String input);

    default String addStars(String input) {
        return "*** " + input + " ***";
    }

    static String staticMethod() {
        return "Static method in interface";
    }
}

StringProcessor processor = s -> s.toUpperCase();
System.out.println(processor.process("hello"));                // HELLO
System.out.println(processor.addStars("hello"));               // *** hello ***
System.out.println(StringProcessor.staticMethod());            // Static method in interface

Q2: Using Supplier and Consumer

Supplier<String> supplier = () -> "Data from supplier";
Consumer<String> consumer = data -> System.out.println("Consumed: " + data);

String data = supplier.get();
consumer.accept(data);

Q3: Using Optional with Function and Predicate

Optional<String> optional = Optional.of("Java");

optional.filter(s -> s.startsWith("J"))
        .map(String::toUpperCase)
        .ifPresent(System.out::println); // Output: JAVA
