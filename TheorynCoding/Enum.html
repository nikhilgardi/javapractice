1️⃣ Your Enum Class: EncashmentStatus

public enum EncashmentStatus {
    INITIATED("I"),
    IN_PROGRESS("P"),
    APPROVED("A"),
    REJECTED("R"),
    INACTIVE("N"),
    PAYMENT_ADVICE("V");

    private final String code;

    EncashmentStatus(String code) {
        this.code = code;
    }

    public String getCode() {
        return code;
    }

    public static EncashmentStatus fromCode(String code) {
        for (EncashmentStatus status : values()) {
            if (status.code.equalsIgnoreCase(code)) {
                return status;
            }
        }
        return null;
    }

    public static EncashmentStatus fromName(String name) {
        try {
            return EncashmentStatus.valueOf(name.toUpperCase());
        } catch (IllegalArgumentException | NullPointerException e) {
            return null;
        }
    }
}

2️⃣ Main Class to Test Enum

public class MainApp {
    public static void main(String[] args) {
        // Accessing enum constant
        EncashmentStatus status1 = EncashmentStatus.INITIATED;
        System.out.println("Enum Constant: " + status1);

        // Get code
        System.out.println("Code: " + status1.getCode());

        // From code to enum
        System.out.println("From Code: " + EncashmentStatus.fromCode("P"));

        // From name
        System.out.println("From Name: " + EncashmentStatus.fromName("approved"));

        // Loop all
        for (EncashmentStatus s : EncashmentStatus.values()) {
            System.out.println(s.name() + " -> " + s.getCode());
        }

        // Ordinal
        System.out.println("Ordinal of APPROVED: " + EncashmentStatus.APPROVED.ordinal());

        // Switch
        switch (status1) {
            case INITIATED -> System.out.println("Status is INITIATED");
            case APPROVED -> System.out.println("Status is APPROVED");
            default -> System.out.println("Other status");
        }
    }
}

3️⃣ Java Enum Interview Questions & Answers

✅ Basic

  • Q: What is an enum in Java?
    A: A type-safe collection of constants.
  • Q: Can enums have constructors?
    A: Yes, but they must be private.
  • Q: Can enums be used in switch statements?
    A: Yes.
  • Q: Difference between name() and toString()?
    A: name() is final, toString() can be overridden.

✅ Advanced

  • Q: Can enums implement interfaces?
    A: Yes.
  • Q: Can enums have abstract methods?
    A: Yes, each constant must override.
  • Q: How do you convert a string to enum safely?
    A: Use try-catch around valueOf.

✅ Tricky

  • Q: Can enums extend classes?
    A: No, they implicitly extend Enum class.
  • Q: Can you create enum instances using new?
    A: No.
  • Q: Are enums thread-safe?
    A: Yes.

✅ MNC-Level

  • Q: Why enums over static final constants?
    A: Type-safe, extensible, and feature-rich.
  • Q: Can enums implement the strategy pattern?
    A: Yes, see example below.

4️⃣ Java Enum Code Examples (Tricky & MNC)

Enum as Strategy Pattern

interface Operation {
    int apply(int a, int b);
}

enum MathOperation implements Operation {
    ADD {
        public int apply(int a, int b) { return a + b; }
    },
    SUBTRACT {
        public int apply(int a, int b) { return a - b; }
    };
}

Enum with Factory Method

enum NotificationType {
    EMAIL {
        public void notifyUser() {
            System.out.println("Sending EMAIL");
        }
    },
    SMS {
        public void notifyUser() {
            System.out.println("Sending SMS");
        }
    };

    public abstract void notifyUser();
}

Enum with Multiple Fields

enum Status {
    ACTIVE("A", "Active User"),
    INACTIVE("I", "Inactive");

    private final String code, description;

    Status(String code, String description) {
        this.code = code;
        this.description = description;
    }

    public String getCode() { return code; }
    public String getDescription() { return description; }
}

Enum in REST API (Jackson)

@JsonFormat(shape = JsonFormat.Shape.OBJECT)
public enum UserRole {
    ADMIN("A", "Administrator"),
    USER("U", "Standard");

    private final String code;
    private final String label;

    UserRole(String code, String label) {
        this.code = code;
        this.label = label;
    }

    @JsonValue
    public String getCode() { return code; }

    @JsonCreator
    public static UserRole fromCode(String code) {
        for (UserRole role : values()) {
            if (role.code.equalsIgnoreCase(code)) return role;
        }
        throw new IllegalArgumentException("Invalid code: " + code);
    }
}
